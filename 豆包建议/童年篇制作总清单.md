# 栖式特调 · 童年篇制作总清单 v1.4

> ⚠️⚠️⚠️ **本文件第一页必须先读——这是豆包生成风格出错的核心原因。** ⚠️⚠️⚠️
>
> **问题根因**：之前给豆包的 prompt 把"风格描述"放在了清单正文里，但豆包生成图片时不会认真阅读长篇清单——它只看 prompt 的前几行。
> **修正方案**：以下"风格强制块"必须作为**每个图片生成请求 prompt 的第一行**出现，不能藏在清单深处。

---

## 🎨 风格强制块（复制到每个 prompt 的最开头）

```
【画面风格锁定 · 必须先读此项，再读具体场景描述】

总体画风：水彩手绘 + 复古滤镜，像 90 年代末 2000 年初的家庭录像带翻出来的画面。
  水彩：颜料在纸上晕开的轻微透明感，边缘不是锐利黑线，而是柔软的色块。
  复古：整体微微泛黄，有轻微颗粒感（像旧胶卷），不使用高对比度和浓烈饱和色。

色调锁死：米黄 (#F5EFE0) × 淡蓝 (#B8D8E3) × 浅粉 (#F2C7D1) × 薄荷绿 (#BED9C6)。
  禁止使用：浓黑、纯红、霓虹蓝、亮黄、任何浓烈饱和色。
  禁止使用：现代数字 CG 锐利线条、赛璐璐风格、3D 建模质感。

小林栖外观锁死（只要小林栖出现，必须严格符合）：
  发色：浅金色 / 白金，柔软微卷短发，留到耳下，发梢常常翘起来
  眼眸：淡蓝色 / 浅灰蓝色，很大很亮，像玻璃珠，睫毛很长
  皮肤：很白，透着点粉
  体型：6~8 岁小孩，比同龄稍高但很瘦，手腕脚踝细细的
  脸上永远有温暖的光斑（太阳光斑、窗格影子、夕阳斜照——必须有！）
  表情：不是"天然呆可爱小孩"——是"漂亮、娇纵、狡黠、知道自己会被照顾"
  常服：白色或米色衬衫/针织衫 + 小蝴蝶结；背着蓝色布制双肩包；书包拉链必须挂着星星流苏挂饰

其他角色锁死：
  童年篇不出现高中篇四位女主（白石雪乃、佐藤健太、西园寺凛、水野真理）。
  只能使用童年篇 NPC（妈妈、外婆、检票员姐姐、护士、街拍师、白发馆长、公鸡等）。

构图偏好：
  从"大人视角"俯拍小林栖，或从"小林栖视角"仰拍大人。
  画面光线必须柔和，阳光是暖金色，不是白色。
```

## 八、常见错误与修正手册

| 错误 | 原因 | 修正方法 |
|---|---|---|
| 小林栖是黑头发黑眼睛 | 直接套用常见"亚洲小孩"设定 | 重新生成——**浅金色/白金头发 + 淡蓝色眼眸** |
| 小林栖是"可爱天然呆"表情 | 误读性格设定 | 他是漂亮、娇纵、任性、狡黠——表情里要有"知道自己漂亮"的那种狡黠 |
| CG 画风和高中篇一样 | 忘记童年篇有独立画风 | 童年篇必须是水彩手绘感 + 复古滤镜（像 90 年代末家庭录像） |
| 小林栖没戴星星流苏挂饰 | 忘记标志性物件 | 蓝色布制双肩包的拉链上必须有星星流苏挂饰 |
| 画面颜色太饱和 | 误选了现代感配色 | 色调必须是米黄、淡蓝、浅粉、薄荷绿 |
| 出现高中篇四位女主 | 误把两条线混在一起 | 童年篇角色无重叠。请从童年篇角色表中选 NPC |
| 小林栖脸上没有温暖光斑 | 漏了最关键的一点光 | 必须有阳光/窗格/夕阳光斑打在他脸上 |

## 九、请求发送指南（强制执行）

### 📌 单个图片的 prompt 模板

> 每次向豆包请求生成图片时，请**完整粘贴以下模板**，替换最后一行的具体描述。
> **风格强制块必须放在 prompt 最前面，不能放在描述后面。**

```
【画面风格锁定 · 必须先读此项，再读具体场景描述】

总体画风：水彩手绘 + 复古滤镜，像 90 年代末 2000 年初的家庭录像带翻出来的画面。
  水彩：颜料在纸上晕开的轻微透明感，边缘不是锐利黑线，而是柔软的色块。
  复古：整体微微泛黄，有轻微颗粒感（像旧胶卷），不使用高对比度和浓烈饱和色。

色调锁死：米黄 (#F5EFE0) × 淡蓝 (#B8D8E3) × 浅粉 (#F2C7D1) × 薄荷绿 (#BED9C6)。
  禁止使用：浓黑、纯红、霓虹蓝、亮黄、任何浓烈饱和色。
  禁止使用：现代数字 CG 锐利线条、赛璐璐风格、3D 建模质感。

小林栖外观锁死（只要小林栖出现，必须严格符合）：
  发色：浅金色 / 白金，柔软微卷短发，留到耳下，发梢常常翘起来
  眼眸：淡蓝色 / 浅灰蓝色，很大很亮，像玻璃珠，睫毛很长
  皮肤：很白，透着点粉
  体型：6~8 岁小孩，比同龄稍高但很瘦，手腕脚踝细细的
  脸上永远有温暖的光斑（太阳光斑、窗格影子、夕阳斜照——必须有！）
  表情：不是"天然呆可爱小孩"——是"漂亮、娇纵、狡黠、知道自己会被照顾"
  常服：白色或米色衬衫/针织衫 + 小蝴蝶结；背着蓝色布制双肩包；书包拉链必须挂着星星流苏挂饰

【具体画面】
小林栖正面露脸（浅金色头发、淡蓝色眼睛），踮脚，鼻尖贴在游乐园检票口玻璃上。玻璃上有他呼的雾气。阳光从头顶打下来，形成光斑。
```
