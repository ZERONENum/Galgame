## 一、音乐制作标准（AI 生成强制规范）

### 1.1 总体音乐风格定位

高中篇的音乐基调：**日系校园 × 温暖回忆 × 少年心事**。

| 维度 | 规范 |
|------|------|
| **核心气质** | 温暖、微甜、带一丝惆怅；像午后阳光穿过教室窗帘的光斑 |
| **音乐语言** | 以原声乐器为主，不使用电子合成音色（除非是特定场景如游戏机厅） |
| **录音质感** | 暖色调、略有房间混响（像在空教室里弹琴），不做干声硬压缩 |
| **速度范围** | 日常场景 70–90 BPM；欢快场景 100–120 BPM；忧伤/回忆场景 55–70 BPM |
| **调式偏好** | 大调为主（温暖明亮）；忧伤场景用自然小调或多利亚调式 |
| **节奏型** | 分解和弦琶音（钢琴/吉他）+ 缓慢的铃鼓/沙锤；避免重鼓点 |
| **音量动态** | 起伏柔和，不做大动态对比；峰值 -6 dB，整体响度 -18 to -14 LUFS |
| **时长** | **2 分 30 秒 – 3 分钟**，可无缝循环（首尾衔接需仔细打磨） |
| **文件格式** | **MP3，320 kbps，44.1 kHz，立体声** |
| **命名规范** | `<分类>_<描述>.mp3` —— 中文名 + 下划线，如 `校园_黄昏.mp3` |

### 1.2 禁止事项（必须遵守）

- ❌ **禁止使用电子合成主奏音色**（synth lead、pad bass、8-bit 音色等）—— 除非是"游戏机厅"等特定剧情场景
- ❌ **禁止大动态**（鼓点突然炸响、弦乐齐爆）—— 这是视觉小说的背景音，不是电影原声带
- ❌ **禁止浓烈的日式演歌/民谣风**—— 太有"地域感"会出戏
- ❌ **禁止带人声**（含哼鸣、吟唱）—— 纯音乐
- ❌ **禁止引用版权旋律**（翻奏流行曲、动漫 OP 等）—— 必须原创

### 1.3 乐器编制指南

| 场景类型 | 主奏乐器 | 辅助乐器 | 氛围乐器 |
|----------|----------|----------|----------|
| **日常/校园** | 立式钢琴（温暖音色）、木吉他（指弹） | 弦乐四重奏（2 小提琴 + 中提琴 + 大提琴，低音量铺底） | 风铃、沙锤、远处的下课铃、窗外风声 |
| **黄昏/回忆** | 三角钢琴（柔和音色）、大提琴独奏 | 低音提琴、泛音吉他 | 夕阳下的风铃、远处蝉鸣 |
| **雨天/梅雨** | 立式钢琴（略带湿润音）、木吉他（滑音） | 单簧管、长笛 | 雨点打窗声（极轻微，作为氛围层） |
| **欢快/文化祭** | 木吉他、尤克里里、口琴 | 铃鼓、沙锤、木琴 | 远处的脚步声、笑声（极轻） |
| **悬疑/紧张** | 钢琴（低音区和弦）、大提琴拨弦 | 木鱼、金属敲击 | 房间共振的低频氛围 |
| **冬日/雪** | 钢琴（高音区清澈）、竖琴 | 长笛独奏 | 远处教堂钟声（极轻） |
| **结局/温暖** | 三角钢琴 + 弦乐五重奏 | 钟琴、竖琴 | 心跳般缓慢的铃鼓 |
| **小游戏场景** | 钢琴 + 木鱼/木琴 节奏化 | 8-bit 音色可以**极少量**点缀 | 轻快铃鼓 |

### 1.4 旋律与和声写作提示

- **日常场景旋律**：五声音阶为主，避免复杂的调式互换；旋律线起伏不超过八度；主旋律可由钢琴和小提琴轮奏
- **回忆/忧伤场景**：多用 ii-V-I 和 iv-V-i 进行；减七和弦作为色彩和弦点缀，不可滥用；主旋律可带"叹息式"的半音下行
- **欢快场景**：用切分节奏和附点音符制造跳跃感；主旋律可用分解和弦骨架填充
- **所有 BGM 的共同要求**：主旋律需可识别、可记忆（约 8–16 小节的完整乐句）；副旋律/和声层需足够简单，避免盖过对话朗读

## 四、AI 音乐生成 Prompt 模板

> 把「制作标准 1.1–1.4」作为每首歌的通用前缀，然后加上下面的**特定场景 Prompt**。
>
> **通用前缀（每首歌都要加）**：
> ```
> 请创作一首日系视觉小说的背景音乐。
> 乐器：以原声乐器为主（钢琴、木吉他、弦乐），不使用电子合成主奏。
> 录音质感：暖色调、略有房间混响、不干声硬压缩。
> 节奏：背景音乐——节奏轻柔，不做大动态对比；起伏柔和。
> 时长：2 分 30 秒，可无缝循环。
> 格式：MP3, 320 kbps, 44.1 kHz, 立体声。
> 主旋律：需要有一个可识别、可记忆的 8-16 小节完整乐句。
> 禁止：人声、版权旋律、浓烈日式演歌、电子合成主奏音色。
> 整体情绪：
> ```
>
> **然后在"整体情绪"后面接下面的特定场景描述**。

## 五、制作完成后检查清单

> 每首 BGM 生成后，对照以下 7 项打勾，有任意一项不满足就重做：

- [ ] **乐器正确**：以原声乐器为主，没有不合场景的电子合成主奏（游戏机厅除外）
- [ ] **情绪匹配**：与上方"情绪描述"一致——听音乐时能联想到对应剧情画面
- [ ] **动态柔和**：没有突然炸响的鼓点或弦乐齐爆；整体起伏在 6 dB 以内
- [ ] **可循环**：首尾衔接平滑，循环播放时听不出断点（尤其 P1-7 夕阳慢循环）
- [ ] **时长达标**：2:30 – 3:00；短曲（P2/N5）30 秒即可
- [ ] **文件规范**：MP3, 320 kbps, 44.1 kHz, 立体声；文件名中文 + 下划线

## 七、与童年篇 BGM 的风格区分

| 维度 | 童年篇 BGM | 高中篇 BGM |
|------|-----------|-----------|
| **核心情绪** | 童年回忆的温柔 + 梦境感 + 旧时光滤镜 | 青春校园的温暖 + 少年心事 + 选择的重量 |
| **主乐器** | 音乐盒、小风铃、竖琴、木钢琴（带"旧"质感） | 现代钢琴、弦乐四重奏、木吉他（更"现在"的质感） |
| **录音质感** | 轻微"老磁带"感 + 暖色调 + 频率响应略窄 | 清晰干净但温暖 + 房间混响 + 全频段平衡 |
| **旋律走向** | 简单五声音阶为主，像童谣 | 更完整的 8-16 小节乐句，有变奏和发展 |
| **和弦复杂度** | 三和弦为主，偶尔七和弦 | 九和弦、挂留和弦可用作色彩，但不过度 |
| **节奏型** | 极慢的分解琶音 | 有轻微律动的分解和弦 + 铃鼓/沙锤 |
| **主题动机** | 独立一套——以"糖盒起源"的音乐盒主题为核心 | 独立一套——以"标题音乐"的主题曲为核心 |

> **注意**：高中篇不引用童年篇主题动机，反之亦然。但**真终章**（finale）可以极轻地在结尾处插入童年篇主题动机——作为"童年的林栖与现在的林栖在同一画面里相遇"的彩蛋。（如果决定这么做，需要在 P0-10 的结尾加 10-15 秒的"糖盒音乐盒"主题变奏）
