# 栖式特调 · 童年篇角色设定 v1.2

> 以 `文档/00_项目总览.md` 为总纲。小林栖是高中篇林栖的童年版本——性格雏形必须一致。

## 一、小林栖（6～8岁，小学一、二年级）

> **样貌必须与高中篇林栖有可识别的继承关系。** 发色、眼色、五官气质必须一致。

| 项目 | 设定 |
|------|------|
| 年龄 | 6～8岁 |
| 身高 | 比同龄孩子稍高一点，但很瘦 |
| **发色** | **浅金色/白金**（与高中篇一致！），柔软微卷短发，留到耳下，发梢常常翘起来 |
| **眼眸** | **淡蓝色/浅灰蓝色**（与高中篇一致！），很大很亮，像玻璃珠；睫毛很长 |
| 皮肤 | 很白，透着点粉 |
| 脸型 | 圆圆的但下巴尖尖，典型的"漂亮小孩" |
| 体型 | 手腕脚踝都细细的，显得很脆弱 |
| 常服 | 白色或米色衬衫/针织衫，领口系一个小蝴蝶结；背着蓝色布制双肩包；**书包拉链挂着星星流苏挂饰**（童年篇起源物件） |

### 特殊场景服装

| 场景 | 服装 |
|------|------|
| 游乐园 | 带条纹的棉质T恤，牛仔短裤，白色帆布鞋 |
| 医院 | 淡蓝色短袖校服，胸前别着小徽章 |
| 商场街拍 | 卡其色针织衫，领口露出白色衬衫领 |
| 美术馆 | 白色短袖衬衫，袖口微微卷起 |
| 外婆家 | 无袖汗衫，短裤，人字拖（很接地气） |

### 性格（高中篇性格的雏形）

| 关键词 | 表现方式 |
|--------|---------|
| **万人迷体质** | 天生有吸引他人关心的气场；阿姨们会主动给他糖，检票员会放他走VIP，馆长会为他打破原则 |
| **娇纵但不讨厌** | 会挑剔温度、糖的甜度，但语气带着撒娇式的抱怨，让人无法拒绝 |
| **任性** | 想做的事一定要做到；用"无辜的眼神+软声软气的要求"达成目的 |
| **恶劣（小孩式的）** | 会"不小心"碰掉东西，会把糖盒随手塞给别人然后转身走，会说"它自己断的" |
| **天真的狡黠** | 知道自己漂亮、知道自己会被照顾；会适度利用这点，但又不全然是恶意 |
| **贪心** | 会收下很多人的好意，但不会"白拿"——会用自己的方式回馈 |
| **孤独** | 爸爸不常在家，妈妈是唯一的温柔；内心比同龄孩子成熟，但又不肯表现出来 |
| **会"还债"** | 这是他的核心行为准则——会接受偏爱，但一定会用某种方式"还"回去 |

### 习惯小动作

- 说话时会轻轻歪头
- 被批评时会盯着自己的脚尖，但脚尖会偷偷左右晃动
- 想达成目的时，眼睛会微微睁大，像两只玻璃珠
- 口袋里永远装着小东西：糖、纽扣、纸条、弹弓子……
- 紧张时会咬下唇（不是害怕，而是"让我想想怎么得逞"）

### 核心台词口吻

| 场景 | 例句 |
|------|------|
| 撒娇请求 | "让我摸摸嘛，就一下下~" |
| 得逞后 | "……它自己掉下来的。"（无辜脸） |
| 挑剔 | "太甜。" / "烫嘴。" / "图案像打翻的调色盘。" |
| 自我辩解 | "我又没扯。它自己断的。" |
| 温柔的时候 | "这个给你。" / "它很软。" |
| 想事情的时候 | "……"（沉默很久，眼睛望着远处） |

## 二、童年篇次要角色

### 妈妈（小林栖的母亲，30岁前后）

| 项目 | 设定 |
|------|------|
| 外貌 | 温柔型美女，长发挽在脑后；手上戴着结婚戒指（爸爸不在但戒指一直戴着） |
| 服装 | 浅色连衣裙，腰间系细带；低跟凉鞋或布鞋 |
| 性格 | 温柔但有原则；会宠孩子但不会溺爱；对林栖的"小诡计"心知肚明，但常常选择睁一只眼闭一只眼 |
| 教育方式 | "可以接受好意，但不可以去交换"——这是林栖"还债"行为准则的来源 |
| 与林栖 | 是林栖唯一的安全感来源 |

### 检票员姐姐（游乐园 · 彩虹入场券章节）

| 项目 | 设定 |
|------|------|
| 年龄 | 20岁出头 |
| 服装 | 游乐园员工制服，有彩虹条纹 |
| 性格 | 善良、爱笑；对可爱小孩毫无抵抗力 |
| 关键行为 | 把工牌流苏给小林栖看，结果流苏"刚好断了" |

### 护士阿姨 × 3（儿童医院 · 眼水经济学章节）

| 项目 | 设定 |
|------|------|
| 形象 | 三个候诊的阿姨，30～50岁不等 |
| 共同特点 | 都有孩子或有孙子 |
| 关键行为 | 看到小林栖含泪但不哭的样子，集体切换到"心疼模式"——塞巧克力、塞果冻、塞玩具 |

### 街拍师（商场 · 橱窗银翼少年章节）

| 项目 | 设定 |
|------|------|
| 形象 | 戴帽子，挂相机，文艺青年样 |
| 性格 | 对"好看的画面"有敏锐的直觉 |
| 关键行为 | 一眼就觉得林栖是天生的镜头宠儿；把纽扣划过射灯的画面拍下来，P成流星 |

### 白发馆长（美术馆 · 釉光小夜曲章节）

| 项目 | 设定 |
|------|------|
| 形象 | 白发苍苍，拄拐杖，穿中式衬衫 |
| 性格 | 表面严肃有原则，内心柔软 |
| 关键行为 | 被小林栖一句"它在发抖"动摇原则，安保系统"刚好"休眠了5分钟 |

### 馆长孙女（美术馆 · 釉光小夜曲章节）

| 项目 | 设定 |
|------|------|
| 形象 | 初中生，扎马尾，戴眼镜 |
| 性格 | 鬼灵精 |
| 关键行为 | 偷偷帮小林栖动了安保系统的手脚，自己躲在角落比心 |

### 外婆（外婆家 · 青铜喙与悬命贴章节）

| 项目 | 设定 |
|------|------|
| 形象 | 60多岁，头发花白但梳得整齐；戴老花镜 |
| 性格 | 慈祥，话不多；对林栖的顽皮很了解，但从不真的生气 |
| 关键行为 | 在林栖和公鸡大战之后，把鸡腿夹进他碗里；养了新的小鸡崽啄手背 |

### 小青（公鸡 · 青铜喙与悬命贴章节）

| 项目 | 设定 |
|------|------|
| 形象 | 青绿色尾翎的公鸡，左爪缺了半趾（林栖去年用玩具车碾的） |
| 性格 | 记仇，勇敢；是童年篇里"最有原则"的角色 |
| 关键行为 | 从梁上飞扑下来叼林栖耳垂上的银钉；在砧板上被褪毛前保持趾高气扬 |

## 三、小林栖的表情设计（供 CG 插画参考）

| 表情名称 | 描述 | 使用场景 |
|---------|------|---------|
| **无辜眼** | 眼睛睁得很大，瞳孔圆圆的，不眨眼 | 犯了错想蒙混过关时 |
| **狡黠笑** | 嘴角一边微微上翘，眼睛弯成月牙 | 得逞后、偷偷摸摸时 |
| **委屈脸** | 下嘴唇微微噘起，眼眶泛红但不哭 | 想要求某事但怕被拒绝时 |
| **认真脸** | 眉头微蹙，眼睛一动不动 | 盯着鼻烟壶、盯着公鸡、盯着灰鸽娃偶时 |
| **"随便啦"脸** | 头微侧，眼睛斜瞟，嘴角不笑 | 被妈妈教育但不打算真改的时候 |

## 四、标志性物件（贯穿童年篇）

| 物件 | 出现章节 | 意义 |
|------|---------|------|
| **星星流苏挂饰** | 彩虹入场券 → 贯穿至羽翼沉柜时 → 高中篇 | 从游乐园工牌上"掉下来"的流苏；书包拉链的装饰；是童年→高中的传承锚点 |
| **薄荷糖铁盒** | 糖盒起源 → 彩虹入场券 → 终章 → 高中篇 | 妈妈第一次给的小铁盒；糖吃完了糖纸还在；是"收藏好意"的容器 |
| **灰鸽娃偶** | 彩虹入场券 → 羽翼沉柜时 | 从游乐园纪念品商店"幸运获得"；小林栖抱着它睡 |
| **弹弓** | 青铜喙与悬命贴 | "罪恶的弹弓"——小林栖用它试图打公鸡，结果打碎玻璃 |
| **柠檬糖** | 眼水经济学 → 贯穿 | 小林栖偏爱柠檬糖；橘子糖会被他"吐掉换掉" |
| **银钉 / 纽扣** | 彩虹入场券、橱窗银翼少年 | 小林栖耳朵上的银钉（被公鸡叼）；从橱窗前获得的纽扣——放进口袋成了"金属流星" |

## 五、CG 画面构图原则（强制执行）

1. **小林栖可以露脸**——童年篇是他唯一允许正面入镜的时期
2. **发色眼色必须一致**——浅金色/白金头发，淡蓝色眼眸
3. **温暖的光**——小林栖脸上永远有温暖的光斑（太阳光斑、窗格影子、夕阳斜照）
4. **构图方式**——常常从"大人视角"俯拍小林栖；或者从"小林栖视角"仰拍大人
5. **色彩**——米黄、淡蓝、浅粉、薄荷绿；不使用浓烈的饱和色
6. **水彩手绘感**——画面有轻微的透明感，像旧照片或水彩画
