## 二、剧情结构

### 高中篇（主线）

```
序章 → 1月 → 2月 → 3月（小游戏） → 4月 → 5月 → 6月（小游戏） → 7月
     → 8月（小游戏） → 9月 → 10月（小游戏） → 11月 → 12月 → 分岐点
     → 四条个人线 → 真结局（未署名的毕册）
```

| 章节 | 主题 | 核心物件 |
|------|------|---------|
| 序章～1月 | 转学生与"漂亮但麻烦"的第一印象 | 糖、豆浆 |
| 2月～3月 | 错题本开端、巴士站事件 | 错题本、糖纸 |
| 4月·晨光特调 | 食堂特殊待遇公开化 | 豆浆、青瓷碗、冰块 |
| 5月·粉笔灰与糖纸 | 利用佐藤但佐藤心甘情愿 | 薄荷糖铁盒、粉笔灰 |
| 7月·雨幕共舞 | 西园寺的"预判"与林栖的"嫌弃式回应" | 透明伞 |
| 薄荷与芦荟 | 多人偏爱的堆积感 | 芦荟胶 |
| 错题玫瑰 | 把错误变成可收藏的美丽物件 | 纸玫瑰、绿萝 |
| 12月·跨年 | 四人把物件放到林栖面前 | 糖盒、豆浆、伞、错题本 |
| 四条个人线 | 白石：记录 / 佐藤：代劳 / 西园寺：预判 / 水野：修正 | 各自物件 |
| 真结局·未署名的毕册 | 四人观察林栖的合订本 | 毕业册、偷拍照、糖纸 |

### 童年篇（后日谈小剧场）

- 解锁条件：通关真结局
- 时间线：林栖 6～8 岁，小学一、二年级
- 与高中篇关系：基本独立，角色无重叠
- 叙事目的：解释林栖习惯与性格的起源——糖盒、薄荷糖、柠檬、对温度的挑剔、不愿欠人情、表面娇纵内心孤独的双重性

章节结构：
```
01 彩虹入场券（游乐园） → 02 眼水经济学（儿童医院）
→ 03 橱窗银翼少年（商场街拍） → 04 釉光小夜曲（美术馆）
→ 05 青铜喙与悬命贴（外婆家公鸡） → 终章 羽翼沉柜时
→ 单元剧场：糖盒起源
```

## 三、核心角色林栖（必须严格遵守）

### 高中篇林栖（16岁）

| 项目 | 设定 |
|------|------|
| 年龄 | 16岁，高中二年级 |
| 身高 | 约170cm，偏瘦 |
| 发色 | **浅金色/白金**，柔软微卷短发，发梢自然蓬松 |
| 眼眸 | **淡蓝色/浅灰蓝色**，清澈，眼神里带着点漫不经心的狡黠 |
| 皮肤 | 很白，手指修长白皙 |
| 服装 | **黑色精致学院制服（西装外套+白衬衫）**；书包拉链挂着**星星流苏挂饰** |
| 性格核心 | **漂亮、娇纵、任性、恶劣、贪心、自私**——但有**天真无邪的表象**和**被宠坏的少年感** |
| 关键行为 | 知道自己会被照顾，会**适度利用**这点，但又**不是恶意** |
| 行为准则 | **会接受偏爱，但一定会以某种方式"还"回去** |
| 习惯 | 口袋/书包里永远有糖；对温度和甜度有精确挑剔；说话带刺但行动偶尔柔软 |
| 口头禅 | "……"（沉默）、"太甜。"、"烫嘴。"、"图案像打翻的调色盘。"、"随便你。" |

### 童年篇小林栖（6～8岁）

| 项目 | 设定 |
|------|------|
| 年龄 | 6～8岁，小学一、二年级 |
| 身高 | 比同龄稍高一点，但很瘦 |
| 发色 | **浅金色/白金**（与高中篇一致！），柔软微卷短发，留到耳下，发梢常常翘起来 |
| 眼眸 | **淡蓝色/浅灰蓝色**（与高中篇一致！），很大很亮，像玻璃珠 |
| 皮肤 | 很白 |
| 服装 | 白色或米色衬衫/针织衫，小蝴蝶结；蓝色布制双肩包；书包拉链开始有**星星流苏挂饰**（童年篇起源） |
| 性格核心 | **漂亮、娇纵、任性、恶劣、贪心、自私、被妈妈纵容**——这是高中篇性格的雏形 |
| 关键行为 | 知道自己漂亮、会被照顾；会用"无辜的眼神+软声软气的要求"达成目的 |
| 行为准则 | 雏形版——收下好意，但会用孩子的方式"还"回去 |
| 习惯 | 口袋里永远装着小东西：糖、纽扣、弹弓子、纸条；紧张时咬下唇；说话歪头 |
| 口头禅 | "不好吃"、"太热了"、"太冷了"、"我不需要"、"……随便你"、"让我摸摸嘛，就一下下~" |

### 样貌设计原则（强制执行）

1. **发色和眼色在童年篇与高中篇必须一致**——浅金色/白金头发，淡蓝色眼眸。这是角色识别的核心锚点。
2. **小林栖可以露脸**（CG插画中），高中篇林栖不做正面立绘（只以背影、侧脸、手部特写、CG中入镜）。
3. **星星流苏挂饰**是童年→高中的传承物件。
4. **糖盒/薄荷糖**是核心标志物，贯穿两个时期。

### 林栖性格写作禁区（强制执行）

- ❌ 不要写成"清冷温柔天然呆"
- ❌ 不要让他完全无意识地被喜欢（他知道别人在看他）
- ❌ 不要写成恶毒反派——恶劣是孩子气的、漂亮的、被纵容的
- ❌ 不要让他主动索取——他是"自然地收下"，而不是索要
- ✅ 他应当是：被观察、被偏爱、被记录、被预判、被修正的那个"漂亮的中心"

## 五、视觉 / 音乐规范

### 背景图 / CG 命名规则（强制执行）

**使用中文可读命名，禁止英文+数字编号。**

- 高中篇背景：`场景_描述.jpg`，如 `教室_晨光.jpg`、`食堂_窗口.jpg`
- 童年篇背景：`童年_场景_描述.jpg`，如 `童年游乐园_检票口.jpg`、`童年医院_注射室.jpg`
- 高中篇 CG：`CG_场景_描述.jpg`，如 `CG_食堂晨光_吹豆浆.jpg`
- 童年篇 CG：`童年_章节关键字_描述.jpg`，如 `童年_彩虹入场券_小林栖踮脚看玻璃.jpg`

### 色彩风格

- 高中篇：日系校园，柔和暖色调，樱花粉、米白、夕阳橙
- 童年篇：**水彩手绘感、复古滤镜**（像90年代末2000年初的家庭录像），米黄、淡蓝、浅粉、薄荷绿
- 童年篇小林栖脸上永远有温暖的光斑

### BGM 风格

- 高中篇：日常轻快、校园氛围、夕缓慢循环、雨天钢琴
- 童年篇：**钢琴+八音盒（温柔怀旧）**、蝉鸣+轻吉他（夏日）、风声+钢琴（冷而温暖）
