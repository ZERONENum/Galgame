# 栖式特调 · 高中篇角色设定 v1.2

> 以 `文档/00_项目总览.md` 为总纲。本文档提供高中篇四位主角与林栖的详细设定。

---

## 一、林栖（主角 / 玩家视角观察的中心）

> 不是玩家攻略的对象——他是"被四人观察、靠近、扰乱"的中心人物。

| 项目 | 设定 |
|------|------|
| 年龄 | 16岁，高中二年级 |
| 身高 | 约170cm，偏瘦，手腕脚踝都细细的 |
| 发色 | **浅金色/白金**，柔软微卷短发，发梢自然蓬松，微微的不听话感 |
| 眼眸 | **淡蓝色/浅灰蓝色**，清澈但有点漫不经心；睫毛很长，低头时会在眼下投小片阴影 |
| 皮肤 | 很白，手指修长白皙，骨节分明 |
| 服装 | **黑色精致学院制服（西装外套+白衬衫）**；不打领带或系得很松；偶尔袖口会卷起来 |
| 书包 | 黑色皮革双肩包，**拉链上挂着星星流苏挂饰**（童年篇彩虹入场券的起源物件） |
| 标志性小动作 | 指尖轻点柜台、歪头、咬下唇、糖盒打开又合上 |
| 气味 | 薄荷 + 淡淡的洗衣液味 |

### 性格层级（从外到内）

1. **表层**（大多数同学看到的）：漂亮、干净、有点冷淡、话不多
2. **中层**（四个主角能看到的）：娇纵、任性、挑剔、会撒娇式抱怨、知道自己好看
3. **深层**（只有在最私人的时刻才显露）：孤独、记着每一份好意、用自己的方式"还债"

### 与四人的关系

- **对白石雪乃**：察觉到她在记录自己的温度/时间；会故意说"太甜"来测试
- **对佐藤健太**：使唤他擦黑板、替自己跑腿；但转头又把薄荷糖铁盒塞进他口袋
- **对西园寺凛**：看穿了她"故意放伞"的预判；嘴上嫌弃图案，却还是把伞往她那边倾斜
- **对水野真理**：知道她试图用规则理解自己；把79分试卷折成纸玫瑰来回应

### 核心台词风格

> "……你说什么？我刚才在发呆。"（真的在发呆吗？还是故意让对方再说一遍？）

> "阿姨，今天豆浆烫嘴。"（不是问句，是陈述——他知道会被特殊对待）

> "粉笔灰会让手过敏哎。"（声音拖得绵软，笑容像狡黠的狐狸——这是他的"请求"方式）

> "……它自己断的。"（童年时把流苏扯断后说的话，高中时依然用同样的语调辩解掉在地上的糖纸）

> "花期比公式长久。"（把错题折成玫瑰时说的话）

> "你扔掉的每颗糖，都是我的许愿星。"（毕业册最后一页糖纸上的字）

### 画面呈现规范（强制执行）

- ✅ 允许：背影、侧脸、手部特写、CG中半侧脸、被阳光模糊的脸、透过玻璃看到的倒影
- ❌ 禁止：正面完整立绘（他的不可接近感来自不被"看全"）
- ✅ 允许在 CG 中入镜：食堂窗口吹豆浆的侧脸、伞下的发梢、窗台插纸玫瑰的手

## 二、白石雪乃（前桌女生 · 温柔治愈系）

| 项目 | 设定 |
|------|------|
| 年龄 | 16岁 |
| 身高 | 158cm |
| 发色 | 深棕色，及腰长发，发梢微卷 |
| 眼眸 | 棕色温柔眼眸，眼尾微微下垂 |
| 服装 | 水手服，裙子长度到膝盖，白色棉质袜，圆头皮鞋 |
| 代表色 | 樱花粉 |

### 性格
- 安静、心思细腻、善于观察他人情绪
- 默默关心别人，不善于表达
- 容易害羞，说话声音轻柔
- 对喜欢的事物非常执着

### 她怎么看林栖
- 记住林栖到校时间精确到分钟，记住他喜欢豆浆温度在 45 度上下浮动不超过 3 度
- 从不说"我喜欢你"，而是用"我记得"的方式靠近
- 会把他随口说的"今天豆浆有点甜"在心里反复嚼三天

### 路线主题：45度与记录
- 核心物件：温度计（她记的是温度，也是他）、纸杯、糖纸
- 林栖对她的"回礼"：某一天豆浆真的是 45 度，杯子上有她的名字缩写

## 三、佐藤健太（体育委员 · 阳光元气系）

| 项目 | 设定 |
|------|------|
| 年龄 | 16岁 |
| 身高 | 178cm，运动型身材 |
| 发色 | 黑色清爽短发 |
| 眼眸 | 棕色明亮眼眸，总是带着笑容 |
| 服装 | 敞开的运动校服外套，里面是白T恤 |
| 代表色 | 橙黄 |

### 性格
- 阳光开朗，什么事都很乐观
- 有点天然呆，容易被林栖牵着走
- 讲义气，朋友有难一定帮忙

### 他怎么看林栖
- 知道林栖说话欠，也知道自己被牵着走，但仍然愿意
- 在他眼里，林栖的"娇纵"是"需要被照顾"的信号
- 保存着林栖给他的糖盒——糖吃完了，但糖纸一张一张叠好

### 路线主题：糖盒与代劳
- 核心物件：薄荷糖铁盒、粉笔灰、擦黑板的布
- 林栖对他的"回礼"：再把糖盒给他一次——里面装着新的糖和一张"谢啦"的纸条

## 四、西园寺凛（同桌女生 · 傲娇清冷系）

| 项目 | 设定 |
|------|------|
| 年龄 | 16岁 |
| 身高 | 162cm |
| 发色 | 黑色及肩短发，发尾整齐 |
| 眼眸 | 深红色锐利眼眸 |
| 服装 | 校服背心+白衬衫，永远系得整整齐齐 |
| 代表色 | 深紫 |

### 性格
- 表面冷淡，看起来不好接近
- 内心温柔，嘴硬心软
- 观察力强，最懂林栖的任性

### 她怎么看林栖
- 会预判林栖的需要——比如雨天把伞放在他能看到的地方
- 最常被林栖一句话戳破心思，然后嘴硬地还回去
- 对林栖的"特殊待遇"最敏感，也最清楚他不是真的冷淡

### 路线主题：透明伞与预判
- 核心物件：透明伞、雨珠、窗台上的绿萝
- 林栖对她的"回礼"：嘴上嫌弃伞的图案，但下雨时自动把伞往她那边倾斜15度

## 五、水野真理（学习委员 · 认真理性系）

| 项目 | 设定 |
|------|------|
| 年龄 | 17岁 |
| 身高 | 165cm |
| 发色 | 深棕色，整齐发型，偶尔扎低马尾 |
| 眼眸 | 黑色，戴细框眼镜 |
| 服装 | 正式校服西装，领带系得一丝不苟 |
| 代表色 | 天蓝 |

### 性格
- 认真理性，做事一丝不苟
- 原则性强，对自己和别人都严格
- 试图用规则理解林栖，但每次都失败

### 她怎么看林栖
- 试图用统计和题目理解林栖——但林栖是"不符合计数规则的答案"
- 在林栖把79分试卷折成纸玫瑰时，她的规则系统第一次动摇
- 最后承认：有些东西不能被计算

### 路线主题：错题与修正
- 核心物件：试卷、错题本、绿萝盆里的纸玫瑰
- 林栖对她的"回礼"：把一张满分试卷折成玫瑰——但还是插在绿萝里（"完美也只是花的肥料"）

## 七、角色立绘规范

### 立绘风格
- 日系 Gal Game 画风
- 上半身立绘（腰部以上）
- 透明背景 PNG
- 统一光源：从左上方来的柔和光线

### 每个角色的表情
- 标准（日常表情）
- 微笑
- 温柔/害羞
- 惊讶
- 认真/思考

### 林栖不做立绘（强制执行）
- 林栖以 CG 形式入镜
- 画面中林栖的脸可以被阳光/玻璃/角度模糊
- 这是为了保持"被观察但不被看透"的氛围
